Welcome to Omni!

Sciter technology enables you, with a bit of HTML, CSS and scripting knowledge, to quickly create amazing desktop UI and Omni is the ideal tool for you developing needs.

This app, as a replacement for Sciter.exe, is equiped with many of the debugging features found in modern browsers for helping you speed up your daily tasks when developing for the Sciter Engine.

For getting started:

  • in the main screen, quickly click and open HTML samples of the Playground, and inspect them with the DOM tools (press F12)
  • check the Library, where you will find meaningful Sciter related content
  • view the presentation that shows an overview of every feature available: view online presentation
